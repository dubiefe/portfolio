Développement d'une application web

Informations complémentaires

En équipe de 7, nous avons conçu et développé une application web destiné à promouvoir le patrimoine français. Pour ce faire, nous avons réalisé une carte interractive avec un aspect communautaire afin de faire connaître les maisons d'illustres françaises.

Ce projet s'est déroulé en trois phases sur plusieurs mois, une phase de gestion de projet, une phase de conception et une phase de développement.

Avant de pouvoir commencer ce projet, nous avons choisi le patrimoine pour notre application : les maisons d'illustres.

Phase de gestion de projet

Pour cette première phase, l'objectif était de nous préparer pour la suite du projet.

Lors de cette phase, j'ai participé à plusieurs parties :

- la définition des besoins fonctionnels et la construction du diagramme de cas d'utilisation;

- la définition et la mise en page des personas;

- la réalisation des permières maquettes en fil de fer.

Phase de conception

Pour cette deuxième phase, l'objectif était de concevoir l'application avant de pouvoir commencer le développement.

Lors de cette phase, j'ai participé à plusieurs parties :

- la création des maquettes en fil de fer;

- la création des diagrammes BPMN pour définir les relations entre les différents utilisateurs;

- la réalisation du flux de transformation des données (ETL);

- la passation d'une ballade cognitive afin d'évaluer la qualité de nos interfaces.

Le dossier réalisé pour cette phase est consultable et téléchargeable ci-dessous :

Phase de développement

Pour cette dernière phase, l'objectif était de développer l'application.

Lors de cette phase, j'ai participé à plusieurs parties :

- la réalisation sur Talend du flux ETL défini précédemment afin de remplir la base de données;

- le développement des interfaces en HTML, CSS et Javascript;

- le développement des contrôleurs en PHP;

- la passation des tests utilisateurs auprès d'utilisateurs répondant à nos peronas.

L'application ayant été hébergée sur un serveur mis à disposition par l'IUT, elle n'est plus disponible. C'est pourquoi j'ai réalisé une vidéo permettant de montrer toutes les fonctionnalités de l'application. Cette vidéo est consultable en suivant ce lien.
